Final Project

Concept

For my final project, I would like to create a Christmas-themed game based on this asset pack. There will be a user controlled snowman that can travel through a neighborhood of houses. The game's objective is to deliver presents to all the houses, and the user does this by completing a minigame at each house. To make the game more challenging, there will be a Christmas ghost that also moves around in the neighborhood. If the snowman touches the ghost, then he loses a life. If the snowman loses all lives, then the game ends.

Snowman (left) and Christmas Ghost (right)

Minigame Features

For the minigames, I would like to challenge myself and incorporate voice control / interaction with the computer mic or color-based collision detection. Additionally, I would like to make this a multiplayer game, where houses with red roofs contain minigames for player 1, houses with green roofs contain minigames for player 2, and houses with blue roofs have multiplayer games that both players must complete (i.e. players either take turns or play together). All minigames will be related to Christmas to tie the holiday theme together. Minigames will be created using states, and will also most likely require classes and object oriented programming to function properly.
